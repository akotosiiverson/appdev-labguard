LabGuard User Management

This page allows you to add additional users to your authentication system.

Note: The authentication system has been updated to work with your existing auth collection structure. You can use this page to add more users if needed.

A new user will be created with the following default values:

{
    "email": "admin@example.com",
    "password": "admin123",
    "role": "admin",
    "displayName": "Administrator"
}

After creation, you can edit user details in the Firebase Console if needed.

Click the button to create an additional user.

After setup is complete, you can return to the login page.
